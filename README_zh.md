# 书签点击统计

[English](README.md) | [中文](README_zh.md)

一个用于统计书签使用频率的 Chrome 扩展。

## 功能特性（简洁）

- 点击统计：自动记录每个书签的点击次数
- 统计信息：总书签数、总点击、平均点击、分类、最热分类
- 搜索与筛选：按标题/URL 搜索；分类、时间范围筛选
- 排序：标题、点击次数、最后访问时间
- 悬停提示：显示完整时间戳
- 数据导出：CSV / JSON
- 语言切换：英文 / 中文

## 使用方法（简洁）

- 点击工具栏图标打开弹窗
- 使用搜索、筛选、排序查看统计
- 使用菜单（⋮）进行导出、重置、语言切换
- 在“最后访问”处悬停查看详情
